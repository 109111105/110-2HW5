# 使用案例
|案例名稱|會員註冊作業|
|------|-------|
|使用案例描述|乘客註冊會員|
|主要參加者|乘客、駕駛|
|利害關係人與目標|讓乘客成功註冊為會員|
|前置條件|需乘客資料|
|後置條件|乘客、駕駛進行登入|
|主要成功情節|1.進入網站 <br> 2.註冊帳密<br>  3.註冊完成<br>  4.成功登入|
|例外情節|帳號密碼註冊錯誤|
|其他需求|無|
